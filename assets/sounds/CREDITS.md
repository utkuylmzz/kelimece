# Ses Efektleri

Kaynak: [Kenney – Interface Sounds](https://kenney.nl/assets/interface-sounds) (CC0 1.0 — kamu malı,
atıf gerektirmez). Orijinal `.ogg` dosyaları iOS uyumluluğu için `.mp3`'e dönüştürüldü.

| Dosya | Orijinal | Kullanım |
|---|---|---|
| key.mp3 | click_003.ogg | Klavye tuş basışı |
| delete.mp3 | back_002.ogg | Harf silme (⌫) |
| invalid.mp3 | error_003.ogg | Geçersiz/eksik kelime |
| win.mp3 | confirmation_004.ogg | Bulmaca kazanıldı |
| lose.mp3 | close_003.ogg | Hak bitti, kaybedildi |
